# 说明

> 采用docsify构建的学习笔记站点

1. 本地安装docsify-cli

```shell
npm i docsify-cli -g
```



1. 启动docsify

```shell
# > docsify serve ./docs

Serving C:\source\owner\gitbook\docs now.
Listening at http://localhost:3000
```

访问地址：[http://localhost:3000](http://localhost:3000)
