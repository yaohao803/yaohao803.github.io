交通运输部4月27日下午召开发布会表示，今年“五一”假期，也就是4月29日至5月3日，七座以下（含七座）小型客车，继续施行免收通行费政策。随着公众出行热情的快速增长，公路客流量可能创下历史新高。

“五一”假期将迎来出行高峰 公安部发出安全预警

发表于 2023-04-27 更新于 2025-06-10

2023-04-27 11:32      来源：央视新闻客户端

“五一”小长假即将来临，当前各地温度回升、气候宜人，正值外出旅游和走亲访友的黄金时节，加之农村地区春耕春种活动繁忙，货运物流持续高位运行，全国将迎来道路出行高峰，交通流量大幅攀升将给道路交通安全畅通带来巨大压力，道路交通安全风险加大。

公安部结合近年来“五一”假期道路交通事故规律特点，对今年“五一”假期道路交通安全形势进行了分析研判，并发出交通安全预警。

据研判，今年“五一”假期，全国道路交通安全主要存在以下突出风险：一是旅游探亲出行集中，旅游景区热点地区道路交通拥堵和事故风险加大。假期群众旅游出行需求增大，同时跨省跨市出游、返乡探亲出行将明显增多。预计假期将出现自驾周边游、跨省游热潮，叠加市区机场车站接驳运输等出行交通流，干线公路、城区客运站场、景区周边道路易发生交通拥堵和交通事故。

二是假期首尾出行流量大，交通违法肇事风险高。预计今年假期前一天（4月28日）下午、首日（4月29日）上午以及尾日（5月3日）下午出行流量出现高峰。高峰期交通拥堵概率大，部分驾驶人争道抢行、加塞穿行，易引发刮擦、追尾事故。从近3年事故情况看，假期首尾两日交通事故死亡占比较大，呈现“两头高、中间低”的态势。每日19至21时事故多发，特别是假期尾日高速免费通行结束前，部分小客车赶时间下高速，超速行驶违法肇事易发多发。

三是农村地区春播春种繁忙，务农务工出行交通安全风险突出。当前农村地区播种移苗等农耕活动繁忙，务农务工群众出行需求大，货车、三轮车、拖拉机违法载人及面包车、摩托车违法超员易发多发，群死群伤事故风险大。

四是物流运输持续攀升，货车肇事风险突出。据有关部门统计，当前高速公路货车日均通行量达750万辆左右，为今年以来最高峰水平。货车昼夜兼程、多拉快跑，疲劳驾驶、超限超载等问题突出，肇事肇祸风险上升，同时假期公路客货车混行交织，交通状况复杂，易发生客货车相撞事故。

五是春夏季节转换身体易疲劳，南方降雨对出行带来不利影响。假期处于春夏季节转换过程，驾驶人易出现“春困夏乏”，影响驾驶安全。据中央气象台预报，“五一”假期降雨增加，特别是江淮、江南、华南及四川盆地、贵州等地多降雨过程，雨天路滑易引发车辆侧翻、追尾和多车相撞事故。

公安部提示，假期自驾出行，请关注有关部门发布的交通路况信息和交通安全提示，合理安排出行时间，提前规划并熟悉路线。驾车时应全程全员系好安全带，杜绝超速行驶、强超强会、分心驾驶等交通违法行为。假期聚餐聚会多，要时刻牢记开车不喝酒、喝酒不开车。高速公路行车，要保持安全车距和车速，雨天行车要降低车速，减少变道超车。注意提前观察高速出口标志标牌，切勿在出口处急变道、急刹车或停车，可以从下一个出口驶离。如发生交通事故，牢记“车靠边、人撤离、即报警”。假期高速公路、城市出入口附近道路拥挤，切勿占用应急车道。驾车前往景区山区，遇急弯陡坡、长下坡、临水临崖道路，要减速慢行，行经农村、山区公路要注意观察道路两侧车辆和行人。景区周边道路车多拥挤，要注意避让行人、规范停车。假期乘车出行，不要乘坐站外揽客拼团“黑车”、超员客车及非载客车辆，乘车时系好安全带，发现违法行为主动举报。生产物流运输，要严格遵规守法、规范经营，货车驾驶人切勿超限超载、疲劳驾驶、违法载人，农村面包车驾驶人切勿超员载客。（总台记者 李珺）

这次降雨过程临近五一假期，有计划出行游玩的公众需密切关注最新预报、预警信息，合理安排出行时间，注意交通安全。

北方地区冷空气活动频繁 气温震荡中上升

眼下，我国中东部地区正在经历一轮强势的回暖过程。不过，未来一段时间，影响我国的冷空气仍然活跃。总体来看，北方受冷空气影响更大，气温波动也比南方更加明显；此外，气温升得多，降得少，整体将呈现出震荡上升的趋势。

其中，27日至29日，受冷空气影响，西北地区大部、华北、东北、黄淮至长江中下游等地先后有4~6级风，阵风7~9级，气温下降4~6℃，局地10℃左右

但在冷空气影响间歇期，气温又会迅速反弹。预计到5月初的时候，除青藏高原外，我国其余大部地区最高气温都能接近或超过20℃，华北、西北地区东部及其以南的多数地方最高气温将超过25℃，部分地区达到30℃左右。

大城市中，呼和浩特昨天最高气温21℃，28日将下滑到13℃，随后再度反弹，5月3日将升至29℃上下，气温起伏波动较大。

南方在暖湿气流助力下，气温也将不断走高，而且波动幅度比北方小很多。但对于西南地区东部以及华南部分地区来说，后期随着降雨强度增大，气温会降到较常年同期略偏低的水平。提醒公众根据气温变化，及时增减衣物，避免着凉感冒。

5月1日起 贵州将全面进入汛期

发表于 2023-04-27 更新于 2025-06-10

2023-04-27 11:29      来源：央视新闻客户端

记者从贵州省2023年防汛防灾新闻发布会获悉，5月1日起，贵州将全面进入汛期。

贵州省防汛抗旱指挥部等相关部门会商研判认为，今年汛期（5—9月）贵州省降水量预计在550~1100毫米，降水总量较常年偏少。主汛期（6、7月）降水集中、极端性强，局地发生山洪、滑坡、泥石流和城市内涝的风险高。7月下旬至8月上旬，将出现阶段性高温，贵州省的东部、北部、南部边缘地区可能有中等到略偏重的伏旱，防汛抗旱形势复杂严峻。

贵州先后出台了《贵州省2023年防汛抗旱工作要点》《贵州省2023年防汛安全隐患排查整治工作方案》等。推动行业部门逐一落实水库、山塘、山洪灾害危险区、地质灾害隐患点、城市低洼易涝点、涉水景区、临河学校和村寨的防汛责任人2.9万多人。在建立、优化多部门常态化联合会商机制、防汛抗旱应急响应联动机制同时，整合水利、气象、自然资源等18个部门监测数据，及时向相关责任人和公众靶向发布灾害预警信息。

今年以来，贵州全省各级已派出9800多个工作组深入山洪灾害易发区、地质灾害隐患点、城市易涝区、防洪重点部位（水库、山塘、尾矿库）、建筑施工工地、沿江临河工矿企业等，开展汛前防汛安全风险隐患排查整治。目前共排查出隐患9146处，整改完成7231处，正在整改1915处。

据悉，今年以来，贵州已开展防汛抗旱应急演练6400多场次，参演人员14.6万多人。已组建各类应急救援队伍831支1.9万人。储备近4亿多元的防汛抗旱救灾物资，部分省级防汛抢险救灾物资拟向重点地区前置。

面对严峻形势，贵州省立足防大汛、抗大旱、抢大险、救大灾，将加强监测预警，突出抓好山洪地质灾害防御、水库和尾矿库安全度汛、城市排水防涝等重点，做好危险区域群众第一时间转移工作，守住防汛安全底线。（总台记者 陈屹 尹平）

大兴机场“五一”假期预计迎送旅客61.5万人次

发表于 2023-04-27 更新于 2025-06-10

2023-04-27 11:29      来源：央视新闻客户端

记者从大兴机场了解到，大兴机场“五一”假期出行旅客吞吐量将呈现增长态势，旅客出行以旅游、探亲为主。根据预测，“五一”假期期间大兴机场预计共执行航班4131架次，预计共迎送旅客61.5万余人次，预计日均航班量达826架次，预计日均客流量将超12.3万人次，出行高峰日预计在4月29日，当日航班895架次、迎送旅客预计将超14.2万人次。

“五一”小长假在大兴机场运营的相关航司国内热门航线有广州、深圳、成都天府、成都双流、重庆、杭州、昆明等。同时，大兴机场也已恢复北京大兴往返首尔、阿姆斯特丹、莫斯科、多哈、塔什干、阿布扎比、吉隆坡等多个国际航线，中国香港、中国澳门等多个地区航线。

针对“五一”期间大客流，大兴机场多措并举做好运输保障工作。重点加强晚进港旅客的交通保障，做好夜间机场巴士运力的补充，固化形成雍和宫、望京两条夜航机场巴士线路，运营至当日国内航班结束后一小时，及时储备夜间出租车运力。通过出租车调度站公众号、各大出租车调度中心等渠道做好车辆募集工作，协调网约车平台引导更多网约车前往大兴机场运营。此外，大兴机场快轨从草桥站抵离大兴机场最快仅需19分钟，旅客乘坐国航、南航、东航从大兴机场出发或到达，均可免费领取大兴机场快轨票。为满足“五一”假期期间大兴机场早出港旅客值机托运需求，草桥城市航站楼已优化调整假期期间开柜时间，从原7:30提前至7:00，每日新增可办理航班20余架次，覆盖东航、南航、厦航、国航、首都航等多家航司的旅客，使更多早出港旅客享受到解放双手、轻松出行的城市航站楼服务体验。

“五一”期间，大兴机场持续优化特殊群体旅客服务保障措施，为特殊群体旅客提供指引、咨询、帮扶等服务，“爱心休息区”“绿色通道”等将为有需要的旅客提供休息、快速通行等服务保障。大兴机场96158服务热线全天候随时为旅客服务，随时满足广大旅客咨询、爱心服务预约等需求，同时，特殊群体旅客也可通过大兴机场App或小程序预约爱心陪伴服务。大兴机场志愿者团队也将做好节假日出行期间旅客帮扶爱心接力工作，助力旅客顺畅出行。应对“五一”期间机场过夜旅客增多情况，大兴机场增派指廊管家加强夜间巡视频次，为老弱病残孕等特殊旅客提供毛毯，送上一份温暖，用心保驾护航。

大兴机场温馨提示广大旅客，节假日期间旅客出行量增多，请旅客及时关注航班变化并做好出行计划，出行前务必留出充足时间，以免耽误行程。如有出行问题可及时拨打大兴机场服务热线“010-96158”。
