侦察搜救、水上救援、固堤抢险……汛期到来多地严阵以待

发表于 2023-05-26 更新于 2025-06-10

2023-05-26 14:51      来源：央视新闻客户端

近期南方多地进入主汛期，为应对可能发生的汛情，军地应急力量开展了防汛救灾演练，进一步提升遂行抢险救援任务能力。

在广西贺州，围绕抗洪抢险中的侦察搜救、水上救援、固堤抢险和城市排涝等课目，贺州军分区联合消防救援支队等多家单位开展抗洪抢险综合演练。为了检验救援分队的极限救援能力，指挥组临机导调，在距离指挥所600米的湖心岛位置设置被困险情。救援人员驾驶水陆两栖车，在洪水中快速搜索前进，发现被困人员后迅速靠近展开救援。随着落水人员被解救上岸，救治组迅速分类受伤人员紧急送往医疗救护站进行救治。

在江苏徐州，一场军地联合防汛抢险演练拉开帷幕，应急分队迅速清点防汛应急装备，结合驻地受灾情况，相继展开抢筑子堤、城市排涝、救援转移等课目，综合检验民兵分队抗洪抢险应急队伍实战能力。

在湖南新化的资江流域，随着现场指挥员一声令下，一艘艘冲锋舟依次驶出，在水面上先后进行了纵队、前三角、360度大旋转等课目训练，全体民兵密切协同、动作规范，重点对冲锋舟操作、水上航行、救援转移、登舟离岸等课目开展练习。

在湖北武汉某水域，军地双方运用高科技手段，组织民兵、退役军人应急救援等重点社会应急力量，开展应急通信、立体侦测、人员搜救等19个课目的演练，提升了防汛抢险组织指挥、快速反应、协同联动能力。

公安部交管局部署2023年全国高考交通管理工作

发表于 2023-05-26 更新于 2025-06-10

2023-05-26 14:51      来源：央视新闻客户端

2023年全国普通高校招生考试（以下简称“高考”）将于6月7日至8日举行（部分省份于9日、10日组织选科科目考试）。今年高考考生人数超千万，再创历史新高，加之高考期间恰逢汛期，极端天气可能引发突发情况。5月24日公安部交通管理局印发通知，部署各地公安交管部门认真做好全国高考交通管理工作。

通知要求，各地公安交管部门要实地踏勘考场周边道路，优化交通组织，完善设施设置。要充分挖掘考场周边停车资源，开辟临时场地，设置停车引导标志。要配合教育、交通运输等部门摸排涉考运输企业和组织集中接送考生的学校，加强安全教育，提醒驾驶员安全驾驶、驾乘人员全程系好安全带。要加强警力部署，增设护学岗，严查考场周边乱停乱放、乱鸣喇叭等交通违法行为，必要时可采取临时交通管制措施。要严管面包车超员、客车超速、货车违法载人等违法行为，严防发生涉考交通事故。对涉及高考用车的轻微交通违法行为，教育警告后及时放行。对发生涉及考生交通事故的，要迅速出警、快速处理，必要时帮助联系车辆转送考生。要依托铁骑设立护考小分队，加强考场周边主要路口和重点路段的巡逻，为考生提供紧急援助服务，开辟“绿色通道”，全力保障考生安全及时应考。要加强与气象、应急、市政等部门的沟通协调，及早掌握雨情、汛情和险情灾情，认真做好恶劣天气交通应急处置准备。

公安部交管局提醒广大考生及家长，应提前熟悉考场周边道路情况，科学选择出行方式及线路；如遇交通拥堵、忘带证件等问题，请及时报警求助，交警将提供紧急援助服务。其他社会车辆如行经考场周边道路，请严格遵守法律法规，做到降车速、禁鸣笛、按序通行，共同营造安全、畅通、宁静的护考环境。（总台央视记者 李珺）

近期，甘肃、河南等地接连发生涉及面包车超员的群死群伤交通事故。

4月29日清晨，王某某驾驶重型半挂车行驶至甘肃酒泉市一村道时，与一辆正在左转的面包车（核载7人，实载18人）侧面碰撞，造成面包车上7人死亡、11人受伤。

呼吸道合胞病毒感染患儿增加 专家详解如何防范

发表于 2023-05-24 更新于 2025-06-10

2023-05-24 17:44      来源：央视新闻客户端

最近，部分医疗机构儿科发热门诊患者增多，很多孩子都感染了呼吸道合胞病毒。呼吸道合胞病毒有什么危害？它与流感和新冠病毒有何区别？家长应该如何科学应对？来看专家的解读。

感染呼吸道合胞病毒的症状主要表现为鼻塞、流涕、咳嗽、咳痰、发热等；呼吸道合胞病毒通过飞沫和密切接触传播，婴幼儿以及体质弱的老年人容易感染。

专家介绍，在北方地区，呼吸道合胞病毒一般在冬春季比较多见。过去三年，由于公众保持了戴口罩、勤洗手的良好卫生习惯，其感染率较低。今年由于公众活动增加，呼吸道合胞病毒的感染率也随之升高，感染高峰也相对延迟。

如何区分流感、新冠和合胞病毒？专家表示，如果家庭当中没有新冠病毒感染者，孩子突然出现发热、咳嗽等症状，呼吸道合胞病毒感染的可能性更大。流感病毒感染症状相对较重，以高热为主，往往出现浑身酸痛、恶心呕吐等症状，而合胞病毒感染的典型症状是咳嗽、喘息等。呼吸道合胞病毒感染为自限性疾病，一般只需要使用常规抗病毒药物以及止咳化痰类的药物，症状大多会在1至2周内消失。感染呼吸道合胞病毒后，免疫力一般只能维持几个月，所以孩子上一年感染过，第二年还有再次感染的可能。预防呼吸道合胞病毒感染，首先要养成良好的卫生习惯。

汛期到来！湖南13000多座水库严阵以待

发表于 2023-05-23 更新于 2025-06-10

2023-05-23 14:25      来源：央视新闻客户端

眼下湖南已经进入汛期，多地迎来持续降雨天气，作为防洪蓄洪的“主力军”，水库在防洪减灾中发挥着重要作用。作为全国水库数量最多的省域，湖南各地正加紧对辖区内水库除险加固，确保水库安全度汛。

在湖南衡南县龙溪桥水库，随着近期降雨增加，库区工作人员增加了对库区水情的监测频次，及时对水库水位、库容、日降雨量等信息进行更新。库区工作人员实行每4人一组24小时值班值守制，对库区内的险工险段进行监测，还坚持每周对闸门机电设备进行维护保养，确保设备正常运行。

截至5月19日，龙溪桥水库水位为115.57米，距正常蓄水位还有11米，防洪库容充足。为应对紧急汛情，库区提前储存了编织袋、彩料布、救生圈、备用电源等防汛物资，做到有备无患。

在湖南郴州，当地最近频繁降雨，多次出现大到暴雨天气。为了确保辖区内水库安全，库区巡逻员每天都会对堤坝的关键防汛点进行逐一检查，并将检查情况上传到全市水库信息管理后台系统，实时更新动态。

记者从湖南省水利厅了解到，湖南目前共有水库13300座，占全国水库数量的1/7，已鉴定为病险水库的3519座。2023年，全省列入除险加固计划的水库有1171座。按照水利部要求，90%以上的病险水库将在今年年底前完成除险加固。（总台记者 宋凯欣 衡阳台 郴州台）
