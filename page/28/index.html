本周中东部开启晴朗升温模式 西部降水发展增多

发表于 2023-05-08 更新于 2025-06-10

2023-05-08 16:15      来源：中国天气网

5月8日，中东部大部将在晴朗中开启新的一周，气温也将小幅回升。与此同时，西部地区降水发展增多，气温持续低迷，需适当添衣保暖。

中东部晴多雨少 西部地区雨雪发展增多

7日，南方降水过程接近尾声，仅在华南等地出现强降雨。监测显示，广西西北部和东南部、广东中南部、福建东南部、四川盆地西南部及台湾岛等地部分地区出现大到暴雨，广东汕尾和惠州、福建泉州、厦门、漳州及四川乐山等局地有大暴雨。

8日开始，中东部天气将转好。不过降水此消彼长，8日-10日，西部地区雨雪天气持续，并逐渐向东发展。中央气象台预计，8日，甘肃南部、青海南部、西藏北部等地部分地区有小到中雪或雨夹雪，部分地区有大到暴雪；四川盆地北部、云南东南部、广西东南部、广东西南部、海南岛、台湾岛等地部分地区有中到大雨，其中，广西东南部沿海、广东西南部、海南岛等地局部地区有暴雨。

9日，甘肃南部、青海南部、西藏北部等地部分地区有小到中雪或雨夹雪，部分地区有大到暴雪；西藏东南部、青海东部、陕西南部、湖南西部、川西高原北部、重庆、贵州北部、广西中部、海南岛等地部分地区有中到大雨，其中，广西南部、海南岛西部局地有暴雨。

10日，甘肃南部、青海南部、西藏北部等地部分地区有小到中雪或雨夹雪，部分地区有大到暴雪；湖南中南部、江西中北部、西南地区东南部、华南北部和西部等地部分地区有中到大雨，其中，广西北部等地部分地区有暴雨。

今年以来，云南雨水稀少，降雨量不足常年同期的一半，持续少雨导致气象干旱发展，预计11日以后云南将迎来较强降水，或扭转当地今年以来降水偏少的天气格局，利于缓解旱情。

冷空气“浇灭”海南高温 局地降幅达8℃ 西部气温持续低迷

气温方面，8日冷空气继续南下，海南高温将迅速退场，多地最高气温单日降幅将达4~8℃，像是海口7日最高气温还有37℃，8日就将降至29℃。

我国西部地区受持续降水影响，8日-10日气温低迷，多地最高气温不足15℃，青海、甘肃等地部分地区甚至在个位数运行。大城市中，兰州、银川8日-10日最高气温普遍不足20℃，较常年同期明显偏低。

但在中东部地区，随着天气转为晴朗，气温也将小幅攀升，12日后，气温将由常年同期偏低转为偏高。预计本周末，华北平原可能会迎来今年首轮高温天气。目前预报时效较远，需保持关注。

“二阳”症状轻？短时间内会出现大量人群感染吗？

发表于 2023-05-08 更新于 2025-06-10

2023-05-08 14:51      来源：科技日报

刚刚过去的“五一”假期，全国各大景点人头攒动。虽然人员密集，但由于天气转暖，戴口罩的人并不多。很多人担心，如此近距离接触，再加上新的毒株出现，会不会再次感染？就此问题，科技日报记者采访了相关专家。

中和抗体仍有效 “二次感染”风险较低

“从监测数据来看，全国病例有增加趋势，但并没有出现大规模聚集性疫情。 ”5月7日，北京大学第一医院感染疾病科主任王贵强在接受科技日报记者采访时说。

此前，中国疾病预防控制中心公布数据显示，我国新监测到XBB.1.16等12种本土重点关注变异株。

对此，首都医科大学附属北京佑安医院感染综合科主任医师李侗曾表示，XBB.1.16变异株是新冠病毒奥密克戎重组毒株的一种，与之前的毒株相比，它多了一些变异位点，使得其免疫逃逸能力、传染性更强。我国之前对疫苗接种后抗体监测发现，3—6个月之后抗体水平会下降。但经历了上一轮疫情，并没有太多人二次感染新冠病毒，主要是因为大家在疫苗接种之后感染的奥密克戎毒株，这样体内抗体水平会更高，保护期也会更长。

“所以对感染过新冠病毒的人群来说，体内抗体仍处在较高水平，目前看二次感染的风险还是比较低的。” 李侗曾说，但距离上次感染已经超过半年时间左右的老年人、免疫力低下人群，发生二次感染的风险仍会增加，因为这部分人群抵抗力下降得会更快，尤其是面对XBB.1.16这种免疫逃逸能力强的毒株。

王贵强也表示，同一毒株二次感染的风险相对是比较低的。但老年人以及有基础病者，本身免疫功能较弱，产生的抗体水平较低，无论是面对新的变异株还是二次感染风险，都是需要重点关注的人群，这类人更应该完成疫苗接种，以降低重症感染率的发生。

因此，专家们都强调，虽然人员流动增加了病毒的传播风险，但更多的是针对“未阳”人群，这类人群尤其要注意防护。

即使“二次感染” 症状也轻一些

目前，根据我国新冠病毒变异株监测数据来看，本土病例仍以BA.5.2和BF.7及其亚分支为主。从动态看，XBB及亚分支增长较快。近期，输入性病例中XBB变异株已占绝对优势，本土病例中XBB变异株占比增长明显。但目前没有因感染XBB.1.16而导致重症入院和死亡增加的报道。与其他奥密克戎亚分支相比，也没有看到疾病严重程度增加的报道。

“虽然有报道说，XBB.1.16导致结膜炎的发生率增高了，但由于一些其他病毒，如腺病毒、流感病毒、肠道病毒等都会引起结膜炎，所以这个毒株会不会引发更多的结膜炎病例，还需要更多的数据支撑，现在下结论还尚早。”李侗曾说。

近日，不少人在网上晒出阳性的抗原，声称“二阳”后仍出现鼻塞、咳嗽、味觉消失等情况，但比第一次症状要轻。

李侗曾认为，在未来两三个月内，此前没有“阳过”的人感染风险依然较高。随着大家感染后的时间延长，而病毒依然在不断变异，其免疫逃逸能力、传染性不断增强，而人们的抵抗力却在下降。因此，再次提醒此前从未感染过的人群，尤其是老年人以及有基础疾病人群要做好个人防护。

“对大部分免疫功能正常的人来说，即使‘二次感染’，症状也会轻一些，不会像第一次那样在短时间内出现大量人群感染。 并且之后的疫情高峰也不会像此前那么明显，可能会呈现一个较缓的波峰。”李侗曾强调。

2023年4月全国自然灾害以风雹、干旱、低温冷冻灾害为主

发表于 2023-05-08 更新于 2025-06-10

2023-05-08 12:51      来源：央视新闻客户端

近日，应急管理部会同工业和信息化部、自然资源部、国铁集团等部门和单位，对2023年4月全国自然灾害情况进行了会商分析。4月份，我国自然灾害以风雹、干旱、低温冷冻灾害为主，洪涝、地质灾害、沙尘暴、地震、雪灾和森林草原火灾等也有不同程度发生。各种自然灾害共造成794.2万人次受灾，因灾死亡7人，紧急转移安置1万人次；倒塌房屋500余间，严重损坏2600余间，一般损坏6.8万间；农作物受灾面积731.2千公顷；直接经济损失54.4亿元。

4月份全国自然灾害主要特点有：

一、风雹灾害影响湖南、贵州等地，沙尘天气过程较常年同期偏多

4月份，全国出现6次强对流天气过程，23省（区、市）332个县遭受冰雹、大风、短时强降雨等强对流天气，造成部分房屋、设施农业受灾，其中，湖南、贵州、江西等地灾害影响较大。风雹灾害造成全国106.9万人次受灾，因灾死亡5人，损坏房屋6.3万间，直接经济损失16亿元。此外，4月份，有6次沙尘过程影响我国，较常年同期偏多。其中，4月9-13日、27-29日过程为沙尘暴级别，4月18-21日达到强沙尘暴级别，造成华北、西北地区局地农作物受灾。

二、西北、华北等地遭受低温冷冻和雪灾

4月份，有5次冷空气过程影响我国，较常年同期偏多。其中，4月21-24日为全国型寒潮过程，降温幅度大，影响范围广，造成西北、华北等地局地果树受冻、设施农业受损。总体上，低温冷冻和雪灾造成山西、甘肃、陕西等11省（区）132.7万人次受灾，农作物受灾面积189.9千公顷，直接经济损失13.6亿元。

三、南方局地出现洪涝和地质灾害，东北凌汛形势平稳

4月份，全国发生4次强降雨过程，32条河流发生超警洪水，洪涝灾害造成湖南、湖北、福建等15省（区、市）52.3万人次受灾，损坏房屋7800余间，直接经济损失6.2亿元。截至5月1日，嫩江、松花江、黑龙江干流均已全线开江，开江期间凌情平稳，没有发生大的险情灾情。此外，广西、贵州等地发生崩塌、滑坡等地质灾害。

四、云南四川等地旱情持续发展，部分地区森林草原火灾频发

4月份，全国大部地区墒情总体适宜，西南部分地区土壤中度以上缺墒。4月中旬以来，重庆、贵州大部、四川东部等地出现明显降水，有效改善土壤墒情，补充库塘蓄水，旱情得到缓解；云南大部、四川南部等地降水仍偏少，加之同期气温偏高，两地旱情持续发展，因旱需生活救助87万人次，农作物受灾面积295.5千公顷。4月份全国共发生森林火灾56起，主要发生在云南、广西、山东、四川等地；发生草原火灾8起，均发生在内蒙古。

此外，4月份，我国大陆地区共发生4级以上地震3次，最大震级为4月7日新疆拜城4.6级地震，未造成灾害损失；4月2日四川会东4.0级地震造成少量房屋轻微损坏。（总台央视记者 唐国荣）

8日起，我国西部地区阴雨频繁，新疆、甘肃、青海、西藏等地高海拔山区还可能出现雨转雪，当地公众需关注雨雪天气对交通出行的不利影响。此外，目前云南持续少雨，多地气象干旱严重，当地相关部门需做好抗旱工作。

近期，受冷空气频繁活动影响，全国大部地区气温低于常年同期水平，8日起，全国大部还将延续气温偏低的天气格局。

7日，在冷空气和降雨的共同影响下，黄河以南地区气温继续下降，南方地区降温更为明显，局地降幅将超10℃。大城市中，南宁6日还出现高温，7日就将断崖式下降至25℃。

8日起，中东部地区将自北向南开启升温模式，但受冷空气频繁影响，气温回升幅度不大，黄淮到江南、华南等地气温将处在较常年同期小幅偏低的水平，最高气温大多在25℃上下，天气相对舒适，较适宜进行户外活动。

4月18日，北京市丰台区北京长峰医院发生重大火灾事故，造成29人死亡。案发当日，北京市公安局丰台分局以涉嫌重大责任事故罪对相关责任人员立案侦查，后对北京长峰医院及施工单位15人刑事拘留，目前该案正在侦查中。

为依法严厉打击危害安全生产刑事犯罪，保护人民群众生命财产安全，最高检对北京长峰医院重大火灾事故案挂牌督办，要求北京市检察机关充分发挥检察职能作用，协同公安机关及有关部门，依法查明各方责任，夯实案件证据基础，依法惩处相关犯罪，维护被害人合法权益；同时，强化溯源治理，助推安全生产风险防范和综合治理。

北京长峰医院火灾事故暴露出安全生产隐患排查整治不到位、安全生产责任不落实等问题，检察机关将持续加大对危害生产安全刑事犯罪打击力度，以落实最高检“八号检察建议”为抓手，督促相关职能部门加强和改进安全生产监管工作，抓早抓小抓苗头、抓制度规范落实、防患于未然，切实保护人民群众生命财产安全。（总台央视记者 程琴）

记者从福建省人民政府防汛抗旱指挥部了解到，5日，福建省北部地区部分出现暴雨，局部大暴雨。6日，南平、三明、宁德和龙岩四市的大部、福州（罗源、连江、闽清、闽侯、永泰）、厦门（同安）、漳州（长泰、龙海）有暴雨，局部大暴雨，24小时累计雨量50～100毫米，局部可达150毫米，最大小时雨量80毫米。根据《福建省防汛防台风应急预案》，福建省防指决定5月6日6时50分启动防暴雨Ⅳ级应急响应。（总台记者 吴燕敏）

贵州铜仁：多地遭遇强风暴雨 消防紧急处置

发表于 2023-05-06 更新于 2025-06-10

2023-05-06 17:19      来源：央视新闻客户端

5日夜间到6日凌晨，贵州铜仁市多地遭遇强风暴雨，地势低洼区域出现内涝，部分道路被倒塌的树木阻断。当地消防人员及时出动，迅速排除险情。

受强降雨影响，在铜仁市玉屏县，当地一处小区因地势较低，门口积水严重，影响人员和车辆出行。消防救援人员到达现场后，发现小区门口积水最深处达1米，停靠在周围的车辆近半个车身被淹没，且水位仍在上涨。根据现场情况，消防人员在做好个人防护措施后，涉水搜寻排水管道，并利用机动水泵排水。经过近2个小时处置，险情被排除。

在铜仁市沿河县，除受强降雨天气影响外，部分地区风力高达9级。大风造成多条道路旁的大树被吹倒，救援人员利用机动链锯、镰刀等工具，对道路上的倒伏树木进行切割，然后搬离。

经过4个小时的紧急处置，受阻道路恢复正常通行。
