内蒙古5月以来大部地区降水偏少 当前干旱面积超五成

发表于 2023-05-11 更新于 2025-06-10

2023-05-11 22:15      来源：央视新闻客户端

据内蒙古气象局消息，5月以来，内蒙古累计降水量在0～40.8（东乌珠穆沁旗乌拉盖）毫米之间，其中呼伦贝尔市中部和南部地区、通辽市东部地区无降水。与常年同期相比，内蒙古降水量分布不均，东部降水量偏少。

内蒙古全区干旱面积56.49万平方公里，占总面积的56.8%。当前干旱发生面积牧区超六成，农区近五成，林区近二成，牧区、农区和林区均以轻旱为主。

据了解，当前牧区干旱面积超六成。目前内蒙古牧草返青推迟，牧草生长受限，气象部门建议加强对草场的监管和调整调度，草原休牧期适当延长，并做好饲草料调度。另外，5月以来东部大部地区基本无有效降水，土壤墒情偏差，内蒙古农区三类墒在五成以上；未来5天东部农区以小雨为主，建议抓住晴好天气推进春播工作，旱情出现地区及时做好抗旱保墒措施。气象部门会及时发布天气预报预警，适时开展人工增雨作业，特别会加大干旱地区人工增雨作业力度。（总台记者 薛东海 祁晓新）

应急管理部：今年以来事故总量同比下降 重特大事故有所反弹

发表于 2023-05-11 更新于 2025-06-10

2023-05-11 16:38      来源：央视新闻客户端

记者从10日应急管理部召开的新闻发布会上了解到，今年以来，截至5月9日，全国事故起数和死亡人数同比下降。但重特大事故时有发生。针对近期生产安全事故主要集中在哪些领域，有哪些共性，暴露出哪些问题，应急部有关负责人进行了回应。

应急部有关负责人表示，今年以来，从发生的事故情况看，一是事故总量下降，据调度快报，截至5月9日，事故起数和死亡人数同比分别下降33.7%、26.1%。二是较大事故下降，事故起数和死亡人数同比分别下降11.5%、3.4%。三是重特大事故有所反弹，主要是传统的高危行业领域事故较为集中，煤矿发生1起特别重大事故，重大事故中建筑业3起、化工2起，道路运输和工贸各1起。表明这些传统高危行业领域仍然是安全整治的重点，另外其他行业还发生多起重大涉险事故；事故直接原因较为集中，河北沧州“3·27”、浙江金华“4·17”两起重大火灾事故，均是违规电焊作业引发，这与去年河南安阳“11·21”特别重大事故如出一辙。另外，近期北京丰台长峰医院、中化集团下属的山东聊城公司相继发生重大事故。

这些事故反映出，一些地区在应对疫情后的经济发展挑战中，人民至上、生命至上的思想理念没有树牢，安全底线红线没有守住，发展和安全两件大事没有统筹好，甚至只重发展不顾安全；反映出一些地区和单位在抓落实上仍有很大差距，安全生产责任悬空，安全隐患排查整治质量不高；反映出一些企业片面追求效益、追求速度，打乱正常的安全生产程序和工艺，安全现场管理混乱，违法违规忘乎所以；也反映出有的安全监管执法“宽松软虚”，与违法违规行为作坚决斗争的精神劲头不足、措施不果断等。当前，正处于大疫之后经济社会发展的关键期、非常期，对安全生产又是一个重大考验。需要各地区、各有关部门和单位，充分认识安全生产的艰巨性、复杂性，立即动起来、紧起来、严起来，按照国务院安委会部署要求，强化责任措施落实，狠抓重大事故隐患专项整治，加快推动安全治理模式向事前预防转型，真正实现长治久安、国泰民安。（总台央视记者 赵雨彤）

福建龙岩：找到3名防汛巡查中落水失联干部 均无生命体征

发表于 2023-05-11 更新于 2025-06-10

2023-05-11 16:33      来源：央视新闻客户端

5月6日12时起，福建龙岩市新罗区遭遇60年来最大短时强降水。7日凌晨1时40分左右，新罗区铁山镇4名镇村干部在防汛巡查中落水失联。事情发生后，当地市、区两级第一时间成立应急处置联合指挥部，并组织各类救援力量730多人，运用无人机、橡皮艇、声呐、机器人等技术设备在沿途区域实行水陆空不间断拉网式搜救工作。

截至5月10日17时许，救援人员先后在事发地下游雁石溪新罗苏坂段、九龙江漳平西园段找到3名失联人员，均无生命体征。经公安部门技术鉴定并组织家属确认，分别为新罗区铁山镇党委宣传委员陈银樱（女，36岁）；铁山镇宣传干部陈舒婷（女，25岁）；铁山镇平林村党支部委员郭镇泉（男，44岁）。

目前，搜救工作仍在全力进行中。（总台记者 陈庚）

华南连遭强降雨侵袭 北方开启升温下周多地迎高温

发表于 2023-05-11 更新于 2025-06-10

2023-05-11 16:32      来源：中国天气网

5月11日至13日，江南南部、华南等地降水持续，其中广东、广西部分地区有暴雨或大暴雨来袭，当地需防范强降雨可能带来的地质灾害及中小河流洪水等灾害。由于降雨频繁，上述地区升温乏力，体感清凉。但在北方，由于雨雪稀少，升温是主旋律，下周初多地将挑战今年以来气温新高，华北、黄淮局地还将迎今年首个高温日。

华南多地将连遭强降雨 累计雨量大 需警惕致灾性

10日，南方新一轮降水过程逐渐展开影响，华南多地遭遇强降雨侵袭。监测显示，10日08时至11日06时，湖南东部和南部、江西中部和南部、广西东北部和南部、广东北部、海南岛等地部分地区出现大到暴雨，广西南宁和防城港、海南岛东部和昌江出现大暴雨，海南琼海和万宁局地特大暴雨。

中央气象台预计，11日至13日，华南部分地区强降雨持续。预计11日，西藏北部、青海南部、四川西北部等地的部分地区有小到中雪或雨夹雪，局地有大雪；西藏东南部、云南中南部、广西东部和中南部、广东北部和西部、海南岛等地部分地区有中到大雨，其中，广西东部和南部沿海、广东西南部、海南岛西部和东部沿海等地局地有暴雨或大暴雨。

12日，四川南部、云南中南部、广西东南部、广东大部和海南岛等地部分地区有中到大雨，其中，云南西南部、广西东南部、广东西南部、海南岛东部和北部等地部分地区有暴雨或大暴雨。

华南将是此轮降水过程的中心，其中两广地区局地累计雨量可达100至150毫米，当地需谨防强降雨的致灾性。需要注意的是，上述部分地区降雨的同时，还伴有短时强降水、雷暴大风等强对流天气，公众需关注预警预报信息，如遇雷电及时进屋内躲避。

北方开启升温模式 南方阴雨频繁

气温方面，11日开始，北方大部将开启升温模式，一直到下周初，华北、黄淮等多地气温将刷新今年来新高，或迎今年首轮高温过程。省会级城市中，郑州12日将会率先冲上30℃，15日还有望迎来今年首个高温日，体验盛夏般的炎热。

而南方多地受阴雨影响，升温幅度有限。江南多地将在14至15日前后冲击30℃，像南京，11日的最高气温仅有22℃，12日至13日将继续走高。在华南地区，由于强降雨频繁，尤其是两广等地清凉在线，最高气温不足25℃。

各市县、各单位要按照《海南省防汛防风防旱应急预案》及其配套手册、《海南省防汛防风防旱总指挥部工作规则》的职责分工以及《进一步实施更加严格的防汛防风行动措施》《关于进一步规范气象预警与应急响应联动工作的通知》的相关要求，做好各项防范应对工作。（总台记者 高军）

一季度北京PM2.5累计浓度为47微克/立方米

发表于 2023-05-10 更新于 2025-06-10

2023-05-10 11:47      来源：央视新闻客户端

北京市生态环境局5月9日表示，今年一季度，北京市细颗粒物（PM2.5）累计浓度为47微克/立方米，与2022年同期相比有所反弹，但与2019年同期相比略有改善，回归到了常态的改善趋势和改善水平。

同时，一季度还面临了气象条件不利、外来沙尘等因素影响，在这种情况下，全市印发实施《北京市深入打好污染防治攻坚战2023年行动计划》，各区、各部门围绕应对气候变化、大气污染防治、水污染防治、土壤污染防治、生态保护五方面、107项任务，加强统筹组织，扎实推进各项任务措施落实。

北京市紧抓重点环节，持续打好蓝天保卫战。面对不利因素，全市坚持问题导向开展专项治理，加强建筑垃圾消纳场、渣土运输车、施工工地（混凝土搅拌站）扬尘问题查处和视频监控平台使用；下发问题线索，精准指导、重点整治；定期通报道路、工地（场站）出口等尘负荷抽测情况；印发城市“揭网见绿”绿地养护技术指南，规范养护见绿地块。

协同治理挥发性有机物（VOCs）和氮氧化物，组织开展VOCs“一厂一策”治理、强制性清洁生产审核工作，顺义区启动汽车制造行业整体清洁生产审核试点。重点石化企业完成低泄漏呼吸阀和紧急泄压阀更换，并完成近40万个密封点泄漏检测修复，进一步减少VOCs无组织排放。持续优化能源结构，相关区启动了燃油锅炉清洁能源改造、农村煤改清洁能源工程。

完善新能源车推广配套政策，发布关于鼓励汽车更新换代消费的方案，乘用车置换新能源小客车最高补贴1万元。全面做好非道路移动机械监管，全市累计编码登记机械11万余台。（总台记者 王胜东 秦智霆）

《北京市食品安全事故应急预案》正式印发实施

发表于 2023-05-10 更新于 2025-06-10

2023-05-10 11:45      来源：央视新闻客户端

近日，由北京市市场监管局牵头制定的《北京市食品安全事故应急预案（2023年修订）》正式印发实施。《预案》分为总则、组织机构及职责、监测与预警、应急响应与处置、总结评估、应急保障、预案管理、附件等8个章节，遵循人民至上、生命至上，统一领导、分级负责，风险监测、预防为主，依法规范、科技支撑的工作原则。一是明确应急处置组织机构及职责。构建了47个单位或相关技术机构分工负责、联动响应的食品安全事故应急处置机制。二是完善应急响应与处置分级响应体系。规范了启动北京市一般、较大、重大、特别重大食品安全事故四级应急响应程序。

《预案》职责明确、要素齐全、程序规范，有明确的应急保障措施，具有科学性、针对性、实用性、权威性，为有效有序应对本市食品安全事故，最大限度减少事故造成的危害，指导和规范全市食品安全事故应对工作，维护人民生命安全和身体健康，提供了专业指导与实施依据。

下一步，北京市市场监管局将做好《预案》的宣贯工作，指导各区制定属地食品安全事故应急预案，进一步明确工作职责。组织开展食品安全事故应急演练，提升组织能力。（总台记者 陈宝善）

10日至13日广西北部、东部将有强降雨强对流天气

发表于 2023-05-10 更新于 2025-06-10

2023-05-10 11:44      来源：央视新闻客户端

记者从广西气象台获悉，受切变线和冷空气共同影响，5月10日至13日，广西北部、东部有一次持续性强降雨并伴有雷暴大风等强对流天气过程，过程累计雨量桂北、桂东及沿海地区有50～150毫米、局地200毫米以上，最大小时降雨量为40～60毫米，需注意防御暴雨及其引发的次生灾害。

预计受切变线和冷空气共同影响，5月10日至13日广西有一次持续性强降雨并伴强对流天气过程。过程累计降雨量：桂北、桂东及沿海地区有50～150毫米、局地200毫米以上，其他地区20～40毫米，最大小时降雨量为40～60毫米。

此次过程影响时间长、范围广、局地累积雨量大，桂林、柳州、河池、玉林等市的部分地区有较高暴雨灾害风险，西河、洛清江等发生中小河流洪水气象风险较高，融安、融水、罗城、鹿寨等地发生山洪和地质灾害气象风险较高。建议各地、各有关部门加强灾害隐患点的巡查排险，防范强降雨可能引发的山洪、地质灾害、中小河流洪水和城乡积涝等次生灾害，并注意防范暴雨对交通运输、城市运行、农业生产等的不利影响。（总台记者 邓君洋）
