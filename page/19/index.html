高考期间南方多降雨 北方有强对流天气

发表于 2023-06-06 更新于 2025-06-10

2023-06-06 17:59      来源：央视新闻客户端

7日，2023年全国高考将拉开帷幕。关注高考期间全国的天气情况。进入芒种节气，我国气温显著升高、雨量充沛、空气湿度大，高温天气频发。中央气象台预计，7日至9日，北方多地高温继续发展，预计8日前后高温范围将达最大。

7日至9日 北方高温继续发展

从5日开始，我国北方多地炎热迅速升级，根据气象监测，河北中部、北京、天津中南部等地部分地区出现35℃以上高温天气，局地达37～39℃。预计7日至9日，北方30℃和35℃以上的范围还将继续扩展，由京津冀一带迅速波及到山东、河南等地，8日前后高温范围可能达到最大，河北南部、河南西部、陕西中部、山东中北部的部分地区都可能出现高温。此外，新疆北部等地从今天开始也将连续出现35～37℃的高温天气，局地最高气温甚至可能达到40℃以上。

而在南方，随着副热带高压南退，长江以南地区雨水增多，高温范围也将明显缩减，但广东、广西、江西和福建等地白天的最高气温仍然大都在30℃以上，并且上述地区夜间的最低气温可能也有26～27℃左右，加上湿度较大，体感上会比较闷热，需继续做好防暑降温。

南方降雨频繁 北方需警惕雷雨“突袭”

降水方面，中央气象台预计，6日我国南方降雨短暂减弱，但7日到9日，新一轮明显降雨又将来袭，北方多地则要警惕雷阵雨天气。

气象预报显示，6日，海南、广东南部沿海、广西东部、福建、浙江南部及云南南部等地部分地区有中到大雨、局地暴雨，海南岛东南部、广东西南部沿海等地局地有大暴雨。7日至9日，冷空气将南下与西南暖湿气流交汇，在长江沿线形成切变系统，新一轮降水也随之上线，江南、华南、云贵高原、四川南部等地将有中到大雨、局地暴雨；此外，从6日开始东北冷涡生成，7日以后，华北北部、东北多地要警惕午后雷雨“突袭”，部分地区还可能同时伴有雷暴大风、冰雹等强对流天气。气象专家提醒，高考期间南方多降雨天气，北方有强对流，建议考生及家长外出随身携带雨具，关注最新的预报、预警信息，提前规划出行时间和路线，注意交通安全。

中国气象局：我国已进入主汛期和灾害多发期

发表于 2023-06-06 更新于 2025-06-10

2023-06-06 15:47      来源：央视新闻客户端

记者从中国气象局获悉，当前我国已进入主汛期和灾害多发期，暴雨、强对流等灾害性天气多发重发，对气象服务提出更高要求。

对此，中国气象局召开视频会议，对近期汛期气象服务工作提出四点要求：

一要有序做好四川省乐山市金口河区永胜乡山体滑坡灾害应急救援气象保障工作。提高思想认识，要把其作为当前紧急且重要的任务，加强气象监测预警，高度关注近期可能出现的降雨、强对流等不利天气对应急救援工作的影响，防止次生灾害发生，最大限度保障人民生命财产安全。

二要加强灾害性天气监测预报预警服务。加强汛期值班值守，主要领导干部要亲自部署，严格落实领导干部带班、值班人员24小时值守制度，保持高度警觉，紧盯天气形势；加强上下联动，进一步强化国—省、省—市—县会商联动机制，主动开展水文、农业气象、人工影响天气等领域的联合会商，提高精准预报能力；及时做好监测预警和气象灾害风险预估，全面提升气象灾害风险定量化预评估水平，做好主汛期灾害隐患排查和防范，努力做到“预警在先、防范在前”；把握当前服务重点，全力以赴做好暴雨、强对流等高影响、灾害性天气监测预报预警，抓紧抓实当前麦收区与全国高考气象服务保障工作。

三要进一步加强部门联动。强化决策服务，做好临灾预警和叫应工作，主动为各级党委、政府快速高效处置提供有效、有用的决策支撑，特别是发布高级别气象预警时，要第一时间叫应各级党委、政府和有关部门负责同志及时采取防范措施；强化与农业农村、水利、自然资源、森林防火等部门的联动，在灾害性天气发生时，第一时间面向社会发布相关预报预警，提醒公众做好相关防御工作。

四要加大灾害性天气科普解读力度，及时向公众传递相关应急避险知识。共建共享科普资源，利用报刊、电视、微博、微信等全媒体渠道将预报信息和科普知识及时传递给老百姓，引导公众知险避险，切实增强公众防范和自救互救能力。（总台央视记者 任梅梅）

湖北十堰：“急救神器”普配到村级卫生室

发表于 2023-06-06 更新于 2025-06-10

2023-06-06 14:43      来源：央视新闻客户端

日前，湖北省十堰市政府出台《十堰市城乡公共场所配置自动体外除颤器实施方案》：配置3200—6400台，主要配置在城乡公共场所300余处、村卫生室1856个、公办中学215所、物业小区1000余家，以及景区、养老机构等处。

自动体外除颤器简称AED，是一种便携式的急救设备，有着“急救神器”的称号。心跳骤停时，在4分钟内实施心肺复苏和AED除颤，大大提高患者生存率。近年来，全国部分城市开始配置AED。十堰市急救中心调研发现，乡村地区首位致死病种为心脑血管疾病，由于距离医院较远，很难及时急救，因此十堰AED重点配置到乡村。

《方案》确立“政府引导、社会共建，科学布局、覆盖重点，部门联动、属地管理”的基本原则。明确配置AED达到100台—200台/10万人，2023年完成40%，2025年全面完成。 为此，各县市区政府相继出台落实方案，明确目标任务与措施。十堰市卫健委制定《十堰市城乡公共场所自动体外除颤器配置管理办法》，依托十堰市急救中心和各县市区120指挥中心，建立全市AED配置应用信息管理和社会查询平台，方便公众查询和使用。全市151个急救中心（站）开展AED急救培训，提升全民自救互救能力。（总台记者 许伟）

中高考倒计时 这份健康提示请查收→

发表于 2023-06-05 更新于 2025-06-10

2023-06-05 19:32      来源：央视新闻

中考、高考临近，近日北京市疾控中心提示考生和家长，要注意预防流行性疾病，以健康的身体和饱满的精神状态迎接人生中的重要考试。

勤开窗通风

保持空气流通是预防流行性疾病的重要手段，建议每天开窗通风不少于3次，每次至少30分钟，如果天气允许可持续开窗。

科学佩戴口罩

考生应加强自身防护，考前减少出入人员密集的公共场所，在乘公共交通工具、进入环境密闭、人员密集场所时，建议规范佩戴口罩。备用考场内如有发热、咳嗽等呼吸道症状或其他乙类乙管呼吸道传染病考生，考场内所有考生和工作人员应佩戴口罩。考生及同住人员罹患传染病时，要及时向学校报告。

注意饮食卫生

考生不吃或少吃生冷食品，不暴饮暴食，蔬菜、瓜果等要洗净，家庭在加工和存放食物时应注意生熟分开。

注意休息、规律作息

考前要按照考试时间合理安排考生起居，保证良好睡眠，适当运动，规律作息，避免熬夜。

调整考生心态

家长要关注考生心理状态，使考生保持身心愉悦，营造温馨、放松的家庭氛围，多与考生交流和沟通，减轻考生心理压力，引导考生积极向上的心态。如有一般心理问题需要寻求帮助，可以拨打北京市心理援助热线：010-82951332获得免费心理咨询服务。（总台央视记者 牟媞媞）

长江流域6月1日全面进入主汛期

发表于 2023-06-02 更新于 2025-06-10

2023-06-02 10:29      来源：央视新闻客户端

6月1日开始，长江流域进入主汛期。受近日长江上游强降雨影响，长江重庆段水位上涨明显。相关部门根据水位探测情况，及时调整航标位置，确保航道安全。

在长江重庆段羊角堡水位监测站，6月1日水位涨到155米，较前一天上涨了1.28米。工作人员介绍，近期长江上游及重庆均有明显的降水过程，长江水位因此一直都在上涨。为了保证航道安全，工作人员加强了监测及巡逻。

记者在长江涪陵段看到，水位上涨，导致江水浑浊，水流湍急，江面上有不少的漂浮物，工作人员正在清理航标船艇钢缆上的漂浮物。

据介绍，每年6月到8月是长江流域的主汛期，水位变化较为频繁。连日来，长江航道部门派出多个作业组，加强数字航道电子巡查和日常航道巡航，按照航道水位信息，对航标进行重新调整布设，保障汛期长江航道安全。

责编：张泽昕
