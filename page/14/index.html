7月3日至4日，重庆市万州区多地遭受暴雨袭击，引发洪涝和地质灾害。国家防总相关负责人立即作出部署，指导帮助重庆全力做好抢险救援各项工作。7月5日早，国家防总派出由应急管理部副部长、水利部副部长王道席带队的工作组赴重庆万州指导防汛抢险救灾工作。

记者从中国铁路成都局获悉，7月4日7时48分左右，受重庆市万州区境内持续强降雨影响，铁路部门封锁万凉铁路万州站至齐岳山间线路进行安全巡查。铁路工作人员发现万凉铁路鱼背山站至罗田站间驷步河铁路桥部分桥墩被洪水冲击受损，造成该铁路桥梁部分垮塌。

生态环境部：上半年推动解决各类涉气环境问题约3.4万个

发表于 2023-06-30 更新于 2025-06-10

在29日召开的生态环境部新闻发布会上，有关负责人通报，1至6月，生态环境部持续开展重点区域空气质量改善监督帮扶，发现并推动解决各类涉气环境问题约3.4万个。

公安部交管局部署开展夏季交通安全整治行动

发表于 2023-06-29 更新于 2025-06-10

2023-06-29 11:01      来源：央视新闻客户端

为进一步加强道路安全管控、净化道路交通秩序，全力为群众平安顺畅出行营造良好道路交通环境，6月27日，公安部交管局召开全国视频会，动员部署各地公安交管部门深入开展夏季交通安全整治行动，重拳打击易肇事肇祸交通违法犯罪行为，攻坚化解各类交通安全突出风险隐患，坚决防范群死群伤交通事故，坚决维护夏季道路交通安全形势稳定。

据了解，行动期间，全国公安交管部门将针对夏季道路交通安全规律特点，突出管控重点，聚焦“两客一危一货”（公路客运和旅游客运车辆、危化品运输车辆、重型货车和轻型货车）和6座以上小客车等7类重点车辆，严管严查超员、超速、超载、疲劳驾驶、酒驾醉驾、违法载人、不按规定车道行驶等12类易肇事肇祸违法犯罪行为，全力以赴防风险、除隐患、保安全、保畅通。

根据行动安排，各地公安交管部门将持续保持严管严查态势，路面巡查管控与精准布控查缉相结合，科学安排勤务部署，严格公正文明执法，强化重点车辆、重点路段、重点时段执法管控。深入推进道路隐患排查治理，坚持动态清零，加强施工路段交通安全管理，加强校园内外隐患排查治理。强化客货运输安全监管，会同交通运输等相关部门，从源头端发力，深化运输企业风险防范，加强货运车辆安全监管，加强旅游客运安全管理，推进危化品运输协同监管。广泛开展交通安全警示教育，线上线下相结合，紧盯“三夏”时节，开展涉农交通安全风险集中预警；紧盯旅游出行，开展客运交通安全集中警示；紧盯自驾出行，开展文明交通集中倡导活动；紧盯学生出行，开展暑假交通安全警示提示。

公安部提示广大交通参与者，夏季是出游出行高峰，交通安全时刻谨记在心。驾车出行要严格遵规守法，提前关注路况、天气，合理安排行程，尽量减少强降雨、台风等恶劣天气期间驾车出行；牢记“开车不喝酒、喝酒不开车”，自觉抵制酒驾醉驾违法犯罪行为；切勿违法超员、违法超载、违法载人，不要超速行驶、疲劳驾驶、分心驾驶。乘车出行要乘坐安全合规车辆，务必全程全员系好安全带，切勿乘坐超员客车、非法营运车辆以及货车、三轮车、拖拉机等非客运车辆。（总台央视记者 李珺）

拧紧“安全阀”！应急管理部：迅速开展燃气安全专项整治

发表于 2023-06-27 更新于 2025-06-10

2023-06-27 17:02      来源：央视新闻客户端

6月26日，应急管理部召开会议，分析近期安全生产形势，研究部署安全防范措施。会议强调，要深刻吸取宁夏银川市兴庆区富洋烧烤店燃气爆炸事故教训，举一反三开展安全生产风险专项整治，加强重点行业、重点领域安全监管，要坚持“四不放过”严肃开展事故调查，深挖彻查事故原因，一体追究地方领导责任、部门监管责任、企业主体责任，同时会同有关方面继续做好善后处置工作。要积极推动燃气安全专项整治。充分发挥国务院安委办职能作用，推动相关部门迅速开展燃气安全专项整治，督促地方逐街逐店排查隐患，严格燃气经营、配送、使用等环节安全监管，加快完善相关法律法规标准，强化物防技防手段，坚决防止重蹈覆辙。

会议强调，当前正值暑期汛期，安全风险加大，要把防范措施落到基层、落到细节，切实保障人民群众生命财产安全。要抓实抓细重点领域安全监管。拉网排查餐饮、娱乐、教育、医疗机构等人员密集场所逃生出口、消防通道堵塞等隐患，加大现场执法力度，公开曝光典型案例，形成有力震慑。紧盯矿山领域采掘失调、灾害治理不到位等重大风险，强化重点地区、重点矿山、重点环节监管监察，对重大隐患抓住不放、一查到底。督促各地落实油气储罐等重大危险源防高温防雷击措施，严格执行检维修、动火作业、高危区域人员聚集等管理制度，把安全责任压实到每个岗位。要在提高排查整治质量上下功夫。强化考核巡查和督导检查的权威性，继续加大通报、发警示函、约谈等工作力度，将专项整治工作情况纳入年度安全生产考核内容。深入企业开展第一责任人履职情况执法检查，对存在问题的该停产的停产、该关闭的关闭，同时加强指导服务，帮助解决问题、提高安全水平。要强化底线思维做好防灾减灾救灾工作。密切关注灾害天气变化情况，及时研判风险，督促地方加强中小河流、病险水库、尾矿库、堤坝围堰等高风险点巡查防守，落实直达基层责任人的临灾预警“叫应”机制，出现险情果断组织群众转移避险。进一步加强野外火源管理、加大巡护力度，有效管控林火风险。（总台央视记者 唐国荣）

筑牢安全防线！各地开展安全培训演练 提升应急救灾能力

发表于 2023-06-26 更新于 2025-06-10

2023-06-26 18:51      来源：央视新闻客户端

针对夏季高温和汛期抢险救援的实际，连日来，国家消防救援局部署各地开展了应急安全培训和实战救援演练，提升防灾减灾救灾能力，筑牢安全防线。

眼下，各地陆续进入汛期，陕西省消防救援总队近日联合多部门在汉中市举行抗洪抢险救援实战演练。演练模拟某地因暴雨造成洪涝灾害险情，消防、气象、医疗等部门组成应急队伍，赶赴指定区域集结。在水漫村庄救援科目中，救援人员利用重载旋翼机悬挂救生担架，飞行至孤岛一侧，实现快速救援被困人员。

近日，安徽亳州消防部门进行了商场市场消防应急灭火救援演练，演练模拟一个商场市场发生火灾，大量浓烟迅速蔓延，多名群众急需转移。

消防员在进行火情侦查、投送救援器材、扑灭明火等多重任务的同时，进入室内救援转移被困人员，最终将被困人员转移到安全地带。

广西钦州日前组织3500多名消防重点单位人员集中开展消防技能实操教学培训和应急逃生避险演练，在实际操作体验中增强他们消防安全隐患自查自纠和应急避险能力，提高社会单位防灾减灾水平。
