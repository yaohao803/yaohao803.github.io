内蒙古呼伦贝尔陈巴尔虎旗东乌珠尔苏木草原火已被扑灭

发表于 2023-05-02 更新于 2025-06-10

2023-05-02 18:14      来源：央视新闻客户端

记者从呼伦贝尔市森林草原防灭火指挥部办公室获悉，5月1日17时20分，根据卫星遥感监测显示，内蒙古呼伦贝尔市陈巴尔虎旗东乌珠尔苏木查干诺尔嘎查有异常高温点，经陈巴尔虎旗森林草原防灭火指挥部核查，301国道附近发生草原火。火场植被主要以针茅草为主，西北风7～8级。

发现火情后，呼伦贝尔市陈巴尔虎旗森林草原防灭火指挥部迅速启动森林草原火灾应急处置预案，组织人员立即赶赴火场。5月2日0时20分，火情已得有效到控制，国道两侧可视范围内无明火，现场累计出动灭火人员1200余人、车辆200余辆、消防水车12辆、防灭火机具300余台进行看守堵截，其中森林消防队伍345人、车辆58辆。

据了解，截至目前，此次草原火明火已被扑灭。现场风力达7～8级左右，风向西南风，气温15℃。此次呼伦贝尔市陈巴尔虎旗东乌珠尔苏木查干诺尔嘎查草原火扑救工作，总体形势可控，现场力量正在分段清理看守火场，确保不复燃。（总台记者 修治国）

中东部升温多地冲击30℃ 5月2日起降雨将至

2023-05-01 17:21      来源：中国天气网

中国天气网讯 5月1日，我国中东部大部地区气温继续上升，西北地区东部、黄淮、江淮等地部分地区最高气温将升至30℃左右，同时昼夜温差拉大，多地昼夜温差可达15℃以上。降水方面，2日至4日，江汉、黄淮、江淮、江南中北部的部分地区有大到暴雨，局部有大暴雨。

中东部气温继续走高 多地昼夜温差可达15℃以上

4月30日，我国中东部大部地区雨水退场，气温回升，监测显示，下午14时，浙江、江西、辽宁等地部分地区较前一天升温超过6℃。

5月1日，中东部大部气温继续走高，最高气温25℃线将向北推进到华北等地，而西北地区东部、黄淮、江淮等地部分地区最高气温还将升至30℃左右，像是西安或将迎来今年首个30℃，济南、郑州等地也将冲击30℃。由于白天升温较夜间更为明显，昼夜温差也会加大，西北、华北、黄淮到长江中下游多地的昼夜温差可达15℃以上，局地20℃以上，大家请适时调整着装，谨防感冒。

2日到3日，随着降雨增多，黄淮、江淮、江汉等地部分地区气温将有所下降，例如郑州，1日最高气温还将有29℃，3日就将降至21℃。

而新一轮冷空气1日至3日将率先影响新疆，多地将出现4～8℃降温，局地10℃以上。在假期结束后的4日至6日，这股冷空气还将影响长江中下游以北地区。

假期后两天江汉江淮等地雨势较强 局部有大暴雨

降水方面，4月30日，我国雨雪天气主要出现在东北、西南、新疆、西藏等地，监测显示，西藏中部、青海南部等地降雪、雨或雨夹雪1～7毫米；新疆伊犁河谷和乌鲁木齐等地降小到中雨或雨夹雪，局地大雨（33毫米）；黑龙江东部、吉林东部等地部分地区降中雨；西藏东部、云南西南部等地部分地区降中到大雨，云南西双版纳局地暴雨（50～68毫米）。

1日，降水影响范围将向东扩展，西北地区东部、山西、河北、内蒙古等地部分地区会有降雨出现，主要以小雨为主。2日至4日，中东部地区将有较强降雨过程，江汉、黄淮、江淮、江南中北部的部分地区有大到暴雨，局部有大暴雨。

具体来看，中央气象台预计，1日，新疆沿天山地区、青藏高原大部有小到中雪或雨夹雪。新疆伊犁河谷、西藏东南部、川西高原北部、四川东部、重庆等地部分地区有中到大雨，其中，西藏东南部等地部分地区有暴雨（50～60毫米）。

2日，新疆沿天山地区和南疆盆地、西北地区中部、西藏西部和中部等地部分地区有小到中雪或雨夹雪。西藏东南部、黄淮西部、江淮西部、江汉大部、湖南北部、西南地区东北部等地部分地区有中到大雨，其中，湖北中部等地部分地区有暴雨（50～90毫米）。

3日，新疆伊犁河谷和南疆盆地、西藏北部和西南部、西北地区中部等地部分地区有小到中雪或雨夹雪，青海西部等地部分地区有中到大雪（5～7毫米）。华北南部、黄淮、江淮、江汉等地部分地区有中到大雨，其中，山东南部、河南东部和南部、安徽中北部、湖北东部等地部分地区有暴雨或大暴雨（100～120毫米）。

气象专家提醒，五一假期最后两天，江汉、江淮、黄淮等地部分地区雨势较强，降雨同时可能伴有强对流天气，公众请注意关注临近预警、预报信息，合理安排行程。新疆、西藏等地1日至3日雨雪天气频繁，需防范降水对交通运输、农牧业等方面的不利影响。

5月份全国自然灾害风险形势发布

发表于 2023-04-29 更新于 2025-06-10

2023-04-29 11:12      来源：央视新闻客户端

近日，国家减灾委员会办公室、应急管理部会同自然资源部、水利部等部门单位召开会商会，对“五一”期间及5月份全国自然灾害风险形势进行会商研判。

综合分析认为，5月份强对流和强降雨天气可能增多增强，江南、华南等地洪涝、风雹灾害风险较高，长江、珠江流域部分河流可能发生超警洪水；西南等部分地区森林火险等级高；南方和西北等局地地质灾害风险较高；云南等地干旱可能进一步发展。“五一”期间冷空气活动较为活跃，南方有两次降水过程，需重点关注洪涝、风雹、森林火灾、地质灾害、大风沙尘等灾害风险。具体如下：

一是洪涝风雹灾害风险。预计5月份，华南东部、华中南部、华东南部降水较常年同期偏多，5月下旬华南地区将陆续进入“龙舟水”时期。江南、华南等地易出现强降雨及雷暴、大风、冰雹等强对流天气，长江、珠江流域部分河流可能发生暴雨洪水，洪涝和风雹灾害风险较高。

二是森林火灾风险。当前仍为春季防火紧要期，预计5月份，东北、华北、西南等地森林火险等级较高，其中河北东北部、内蒙古东部、辽宁西南部、四川西南部、云南西北部的部分地区森林火险等级高，四川西南部、云南西北部的局部地区可能会出现极高火险等级时段。

三是地质灾害风险。受强降水等因素影响，浙江、安徽、福建、江西、湖北、湖南、广东、广西、重庆、四川、贵州、云南、陕西、甘肃、青海等局地发生地质灾害的风险较高。

四是干旱灾害风险。预计5月份，西南地区南部气温较常年同期偏高1-2℃，降水偏少2-5成。云南等地前期干旱持续时间较长，受气温升高和降水持续偏少影响，干旱进一步发展的可能性较大，需全力做好人畜饮水保障工作。

五是农业灾害风险。4月下旬，受大风和降温降雪天气影响，西北局地出现果树等经济作物受冻、农业设施受损。西南部分地区缺水缺墒造成在田作物受旱，并影响适时春播。目前距小麦大面积收获还有40多天，5月份气温总体偏高，可能遭遇干热风。干旱可能影响云南、贵州西南部和四川南部等地水稻泡田栽插和在田作物生长发育。

六是“五一”期间灾害风险。据气象部门预测，“五一”期间冷空气活动较为活跃，江南、华南等地有两轮降雨过程，主要集中在4月29-30日、5月2-5日。江南、华南及西南地区东部降水量较常年偏多3-8成，局地偏多1倍左右，并伴有强对流天气，暴雨区内部分中小河流可能发生超警洪水，洪涝、风雹灾害风险较高。东北、华北、西南等部分地区森林火险等级较高，四川西南部、云南中北部局部地区森林火险等级极高。浙江、安徽、福建、江西、湖北、湖南、广西、重庆等局地强降雨引发地质灾害风险较高。北方地区大风和沙尘对交通出行和户外活动可能有不利影响。

“五一”假期北方气温起伏较大 5月2日南方迎降水

发表于 2023-04-29 更新于 2025-06-10

2023-04-29 11:11      来源：中国天气网

中国天气网讯 4月29日，“五一”小长假正式开启，我国大部地区雨雪稀少，适宜出行，北方大部沙尘天气明显减弱，但黄淮等地仍有扬沙或浮尘，公众外出需注意防护。5月2日开始南方新一轮降雨又将铺展开来，当地公众需关注临近预警预报信息。

“五一”假期北方气温起伏较大 南方升温为主多地冲击30℃

4月29日，“五一”假期正式开启，今年假期期间，冷空气活动仍频繁，北方大部气温起伏较大，总体呈现在波动中上升的趋势。省会级城市中，像银川，4月28日最高气温仅12.4℃，5月2日就将升至30℃。哈尔滨4月30日的最高气温将仅有10℃，5月2日将升至23℃，隔天如换季，公众薄厚衣物都需备好，及时增减衣物，谨防感冒。

南方受冷空气影响较弱，各地气温将稳中有升，“五一”假期后期，南方最高气温30℃+的区域将迅速增多，四川南部、云南一些地方会有高温出没。像长沙，5月3日最高气温可升至30℃，外出可感受到初夏的热感。

假期首日我国大部雨雪缩减 5月2日南方新一轮降水再来

4月28日，我国雨雪增多，南方降雨达本轮过程鼎盛，其中，四川中部、贵州南部、湖南南部、江西中北部、浙江南部、福建西北部、广西中北部等地部分地区降大到暴雨，湖南衡阳、江西吉安、广西河池和桂林等局地大暴雨（100～212毫米）；西藏中部、内蒙古西部和东北部、山西北部、河北西北部等地部分地区降雪、雨或雨夹雪1～5毫米，局地6～9毫米；北京、河北北部、天津北部等地部分地区出现中雨，北京东北部局地大雨或暴雨，部分地区伴有雷暴大风和冰雹天气。

29日，我国大部雨雪稀少，较强降水主要出现在东北、华南等地。中央气象台预计，新疆南疆盆地南部、西藏中东部、青海西南部、内蒙古东北部等地部分地区有小到中雪或雨夹雪。东北地区大部、西藏东南部、四川南部、云南、江南中东部和南部、华南大部、海南岛、台湾岛等地部分地区有小到中雨，其中，吉林东部、福建东北部、广西东部和南部、广东西部等地部分地区有大雨（25～45毫米）。

30日，新疆沿天山地区和南疆盆地南部、西藏东部和南部、青海大部、川西高原北部等地部分地区有小到中雪或雨夹雪。新疆伊犁河谷、西藏东南部、东北地区东部、西南地区大部、海南岛、台湾岛等地部分地区有小到中雨，其中，西藏东南部等地局地有大雨（25～40毫米）。

总体来看，“五一”假期期间北方大部雨雪较少，天气适宜出行。4月30日南方降水也将明显缩减，但5月2日开始，新一轮降雨又将铺展开来，2日至3日，长江中下游等地将出现强降雨并伴强对流，公众需关注临近预报，返程路上行车保持车距，注意交通安全。

此外，需要提醒的是，4月29日我国沙尘天气虽然范围缩小、强度减弱，但新疆南部、内蒙古中部、河南、安徽、湖北、湖南北部等地的部分地区仍有扬沙或浮尘天气，正值“五一”出行高峰日，公众外出需注意防护。
