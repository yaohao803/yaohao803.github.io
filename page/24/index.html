13日广东云南仍有较强降雨 中东部炎热持续升级多地冲击高温

发表于 2023-05-13 更新于 2025-06-10

2023-05-13 19:22      来源：中国天气网

中国天气网讯 5月13日，广东、云南等地仍有较强降雨；14日华南降雨减弱，云南中西部降雨显著。气温方面，未来几天，中东部大部炎热升级，15日至17日，北京、天津、河北、河南、山东等地部分地区或现高温天气。

14日起华南降雨减弱云南中西部仍有明显降雨

12日，降雨主要出现在华南和云南东南部等地。监测显示，12日08时至13日06时，云南南部、广西东南部、广东西部、海南岛东部和南部等地部分地区出现大到暴雨，海南文昌、琼海等局地大暴雨，万宁局地特大暴雨；另外，河北东部、山东北部等地部分地区出现中到大雨。

中央气象台预计，13日华南等地的降雨天气还将持续；14日，华南降雨将明显减弱，但云南中西部地区仍有明显降雨。具体来看，13日，云南、华南南部和海南岛等地的部分地区有中到大雨，其中，云南西南部、广西东南部、广东西南部、海南岛东部等地部分地区有暴雨或大暴雨。

14日，新疆北疆北部、西藏东南部、云南西部、广东中南部沿海、台湾岛、海南岛等地的部分地区有中到大雨。

目前，云南大部仍存在不同程度的旱情，这轮降雨对于缓和当地气象干旱十分有利。同时，公众出行需密切关注天气预报预警信息，注意交通安全。

中东部炎热升级北方10余省会级城市气温或创新高

未来几天，中东部地区受暖脊控制，主打晴天升温。预计15日前后，气温将到达近期高点，中央气象台预计，15日至17日，北京南部、天津、河北中南部、河南中西部、山东北部等地有35℃高温天气，山东北部部分地区气温可达37℃以上。

13日，升温显著的地区集中在安徽、江苏到江西、福建一带，升温幅度普遍在4℃左右。14日，中东部大部气温继续回升，最高气温30℃以上的地区将会激增，河北中南部、北京、山西南部、陕西大部一直到长江一线都将加入“30℃群聊”，湖北西部、河南北部等地最高气温或达35℃左右。

15日，中东大部气温继续升高，河北北部、北京以及江南多地升温幅度会在6℃左右，最高气温30℃以上的范围向北扩大至吉林西部一带。大城市中，哈尔滨、长春、沈阳、呼和浩特、太原、北京、天津、石家庄、西安、银川、济南最高气温都可能创下今年来新高。

值得注意的是，京津冀、山东北部、河南中西部、湖北西部等地还将出现今年首轮较大范围高温天气。提醒公众干热天气外出时需注意防晒、补水，避免中暑。

5—10秒！陕西将实现地震“秒级预警”

发表于 2023-05-12 更新于 2025-06-10

2023-05-12 17:01      来源：央视新闻客户端

在全国防灾减灾日来临之际，陕西省地震局举办“防震减灾‘三秦行’活动”，向社会公众科普防灾减灾知识，介绍即将上线试运行的陕西省地震烈度速报与预警系统。

走进陕西地震预警中心，几十块电脑、电视显示屏上分布着密密麻麻的数据和图表、图片，这是陕西省地震烈度速报与预警系统中的一个子系统——“数据处理系统”。除了该板块外，整个系统还包括其他四大板块相互配合支持，分别为：台站观测系统、通信网络系统、紧急地震信息服务系统、技术支持与保障系统。根据中国地震局统一安排，陕西省地震烈度速报与预警系统将于5月底启动试运行，试运行结束后，将正式向社会和公众提供地震预警信息服务。

陕西地震预警中心主任赵韬介绍，陕西地震烈度速报与预警系统具有“地震烈度速报”与“地震预警”两大作用。在“地震烈度速报”方面，地震发生以后，救援人员需快速获知何地受到的破坏是最大的，此前，地震发生后的烈度图约需7至10天制作完成。新系统建立之后，烈度图制作速度大幅提升。由于全省建设了1210个观测站点，每10公里一个，地震发生后就能快速估算当地的地震烈度。一般情况下，震后30分钟内，系统即可发布自动烈度速报信息；30分钟—24小时给出震源过程的分析结果和灾害快评估结果。在“地震预警”方面，就目前的技术条件而言，民众接收到的是震后2分钟由机器自动发布的自动地震速报信息，以及震后5—8分钟后来自中国地震台网人工正式发布的速报信息。而地震预警系统上线后，民众将收到“秒级预警”。在破坏性地震来临时，在首台观测仪器被触发后，5—10秒内系统即可给出地震预警信息，通过通信网络和服务终端将发布给用户。（总台记者 刘涛）

华南局地仍有大暴雨 中东部热度逐渐升级

发表于 2023-05-12 更新于 2025-06-10

2023-05-12 17:01      来源：中国天气网

中国天气网讯 5月12日至13日，华南地区的较强降雨仍会持续，局地有暴雨或大暴雨；另外，12日京津冀、内蒙古等局地需防范雷雨天气。气温方面，未来一段时间，中东部地区热度升级，15日前后华北平原或迎来今年首轮高温天气。

华南云南等地仍有较强降雨 华北局地警惕强对流

11日，我国的降水主要集中在华南以及东北等地。监测显示，11日08时至12日06时，广东北部和西部、广西东南部、云南南部、海南岛西部和东北部等地部分地区出现大到暴雨，广东茂名和湛江、广西玉林和北海、海南文昌等局地大暴雨。

12日至13日，华南地区仍有较强降雨，广东、广西、海南、云南等地局地有暴雨或大暴雨。具体来看，中央气象台预计，12日，西南地区南部、华南南部和海南岛等地部分地区有中到大雨，其中，广东西南部沿海、海南岛大部等地部分地区有暴雨或大暴雨。此外，内蒙古中东部、东北地区、京津冀、山东北部等地的部分地区将有小到中雨或雷阵雨，局地有雷暴大风或冰雹天气等强对流天气。

13日，西南地区南部、华南中东部等地部分地区有中到大雨，其中，云南西南部、广东西南部沿海、海南岛东北部等地部分地区有暴雨。

气象部门提醒，华南本轮降雨过程持续时间较长、暴雨区域集中且部分地区短时雨强大，其中海南岛局地累计雨量可达150～200毫米，建议当地公众加强防范城乡内涝、农田渍涝以及中小河流洪水、滑坡、泥石流等次生灾害。另外，12日京津冀、内蒙古等地仍多雷雨，局部或伴有强对流天气，上述地区公众需密切关注临近预报预警信息，出行注意交通安全。

中东部热度逐渐升级 下周初华北平原或迎今年首轮高温天气

12日至14日，中东部地区主打晴天升温，午后的热度也将显著升级。预计15日前后，中东部气温将达到近期高点，黄淮及其以北地区多地将创今年来新高。

从前后对比来看，11日，中东部地区最高温超过30℃的站点还寥寥无几；预计14日，华北中部到长江沿江将有大片地区最高温超过30℃；15日，河北、河南、山东等局地还将有高温天气现身，内蒙古东南部和辽宁西部等局地也将挑战高温；16日，随着雨水增多，北方炎热明显缓和，30℃以上的范围将明显缩减。

大城市中，预计15日前后，石家庄、济南、郑州都有可能挑战今年的首个高温天气，且都明显早于常年。

这轮升温浪潮还将推动长江下游地区到华北平原的入夏进程，夏天的版图有望迅速向北推进，但具体入夏日期还需之后判定。

应急管理部公布4起安全生产举报典型案例

发表于 2023-05-12 更新于 2025-06-10

2023-05-12 17:01      来源：央视新闻客户端

国务院安委会安全生产十五条措施出台以来，各地进一步推动安全生产举报工作，鼓励社会公众参与举报安全生产重大风险、事故隐患和违法行为，发动安全生产社会监督力量。近期，应急管理部从各地严肃查处的举报案件中选取并公布4起安全生产举报典型案例。

案例一：举报某铸造企业重大事故隐患和违法行为

2022年12月，山东省应急管理厅接到群众网络举报，反映菏泽市辖区内某铸造企业存在未配备专职安全生产管理人员、从业人员未经安全生产教育和培训合格上岗作业等多项安全生产违法行为。按相关程序，菏泽市应急管理局对举报内容进行核查。经查，该企业存在从业人员超100人未按规定设置安全生产管理机构或配备专职安全生产管理人员、从业人员未经安全生产教育和培训合格上岗作业、未建立健全安全生产风险分级管控制度、未按规定制定生产安全事故应急救援预案等违法行为。上述行为违反了《中华人民共和国安全生产法》第二十一条第三、五、六项，第二十四条第二款，第二十八条第一款，第四十一条第一款和第八十一条规定，按照《中华人民共和国安全生产法》第九十四条第一款，第九十七条第一、三、六项，第一百零一条第四项等有关规定，菏泽市应急管理局对该企业及其主要负责人分别处罚款人民币19万元和4万元，对发现的问题隐患责令企业限期整改完成，并通报有关情况至同类型企业，全面排查类似问题，形成有效震慑。

举报人发现并举报生产经营单位重大事故隐患和违法行为，根据《山东省安全生产举报奖励办法》第三条第二款规定，山东省应急管理厅于2023年3月予以奖励人民币11.7万元。

案例二：举报未经许可非法经营危险化学品

2022年12月，浙江省瑞安市应急管理局接到群众电话举报，反映瑞安市潘岱街道一房屋内非法储存危险化学品。按相关程序，瑞安市应急管理局对举报内容进行核查。经查，某涂料店在未持有危险化学品经营许可证以及其他危险化学品生产、储存或带储存经营的许可证的情况下，于2022年11月将经鉴定属于危险化学品的某品牌涂料稀释剂（香蕉水）、某品牌涂料稀释剂（天那水）、某品牌松香水等储存在该房屋内并进行经营销售。上述行为违反了《危险化学品安全管理条例》第三十三条第一款规定，按照《危险化学品安全管理条例》第七十七条第三款规定，瑞安市应急管理局责令当事人停止经营活动，没收违法经营的危险化学品和违法所得人民币2104元，处罚款人民币10.1万元。

举报人发现并举报安全生产违法违规行为，根据《瑞安市安全生产重大事故隐患和非法违法行为举报奖励办法》第九条第一项规定，瑞安市应急管理局于2023年2月予以奖励人民币1.5万元。

案例三：举报特种作业人员未持证上岗作业

2022年10月，四川省成都市龙泉驿区应急管理局接到群众走访举报，反映龙泉驿区某汽车制造公司有部分焊工人员无证上岗作业。按相关程序，龙泉驿区应急管理局对举报内容进行核查。经查，该公司焊接特种作业人员未持证上岗作业。上述行为违反了《中华人民共和国安全生产法》第三十条第一款规定，按照《中华人民共和国安全生产法》第九十七条第七项规定，龙泉驿区应急管理局对该公司处罚款人民币6.1万元。同时，查处过程中还发现该公司存在使用危险物品未采取安全可靠措施等其他违法问题，一并依法予以处理。

举报人发现并举报生产经营单位安全生产违法行为，根据《成都市安全隐患举报奖励办法》第九条第一项规定，龙泉驿区应急管理局于2023年2月予以奖励人民币9150元。

案例四：举报某矿业公司瞒报事故

2022年10月，福建省应急管理厅接到群众电话举报，反映龙岩市连城县某矿业公司于2022年10月19日下午发生生产安全事故，造成1死1伤，无单位对事故进行调查和处理。按相关程序，国家矿山安全监察局福建局对该起事故进行提级调查。经查，该矿业公司于2022年10月19日下午发生一起死亡2人的生产安全事故，并隐瞒不报。上述行为违反了《中华人民共和国安全生产法》第八十三条等有关规定，依据《中华人民共和国安全生产法》第一百一十四条等有关规定，国家矿山安全监察局福建局对该公司罚款人民币398万元，由安全生产许可证发证机关依法撤销安全生产许可证，另有24名事故责任人员受到严肃处理。

举报人及时发现并举报生产经营单位瞒报事故行为，根据《安全生产领域举报奖励办法》第十一条第二项规定，福建省应急管理厅于2023年2月予以奖励人民币6万元。

据了解，应急管理部将进一步加大工作力度，继续鼓励社会公众参与安全生产举报，消除重大风险隐患，打击非法违法行为，在全社会营造人人参与安全生产监督的良好氛围。（总台央视记者 唐国荣）

公安部交管局部署推进农村交通安全管理工作

发表于 2023-05-12 更新于 2025-06-10

2023-05-12 17:00      来源：央视新闻客户端

5月11日，公安部交通管理局在江苏宿迁召开全国农村交通安全管理工作现场会，部署各地深入贯彻国务院安委办《关于切实加强农村道路交通安全工作的通知》要求，积极推动党政领导下的综合治理、协同治理，坚持以现代化治理方式破解深层次难题，全力为农村群众出行创造良好道路交通环境。

据统计，2022年以来，各地公安交管部门与交通运输、农业农村、乡村振兴、保险等部门积极推动农村交通安全协同共治，取得良好成效，在农村地区机动车保有量、驾驶人数量均年增1100多万的情况下，一次死亡3人以上较大交通事故同比下降18.2%，未发生重特大交通事故。

公安部交管局相关负责人指出，农村交通安全仍处在爬坡过坎期，道路基础条件较差、车辆安全性能较差、群众交通安全意识较差、管理力量薄弱问题依然是最大短板。特别是随着乡村振兴全面推进，县域城镇化、农业现代化进程加快，快速增长的农村交通要素与传统的管理模式和效能之间的矛盾日趋尖锐，城乡高度融合衍生的农村出行新态势更加复杂，通勤化和务农务工集体出行等特征更加明显，农村道路交通安全形势的严峻性、复杂性依然突出，群死群伤事故阶段性、局部性多发的局面依然严峻，迫切需要通过高水平治理、高质量发展来应对农村交通安全领域的新形势新挑战。

下一步，各地公安交管部门要深入推进农村交通安全责任体系、协同共治力量、劝导管理模式、隐患治理机制、交通安全文化“五大建设”。（总台央视记者 李珺）

山西对两起较大火灾事故查处实行挂牌督办

记者从山西省安委办了解到，5月7日，山西省吕梁市临县临泉镇湫河花苑小区1号楼发生一起较大火灾事故，造成5人遇难。5月8日，山西省吕梁市汾阳市钟楼佳苑三期7号楼发生一起较大火灾事故，造成3人遇难，20人受伤。根据有关规定，山西省安委办决定对这两起事故查处实行挂牌督办。

针对华南地区强降雨过程 国家防总启动防汛四级应急响应

发表于 2023-05-12 更新于 2025-06-10

2023-05-12 17:00      来源：央视新闻客户端

据气象部门预报，5月11日至13日，华南中部和南部的部分地区有大到暴雨，局地大暴雨，上述部分地区伴有短时强降水，局地有雷暴大风等强对流天气。此外，受孟加拉湾风暴“穆查”影响，西藏东南部及云南局部将有暴雨或大暴雨。根据当前防汛形势和《国家防汛抗旱应急预案》有关规定，国家防总于5月11日18时启动防汛四级应急响应，派出2个工作组分赴海南和广东、广西指导强降雨防范应对。

5月11日，国家防总办公室、应急管理部与水利部、中国气象局会商研判，视频连线广东、广西、海南等省（自治区）防办，研究部署防汛工作。会商要求，要密切监视天气变化，加强监测预警和会商研判，及时启动、调整应急响应，督促地方落实预警响应联动和临灾预警“叫应”机制，切实加强道路交通、涉山涉水景区、在建工地、危旧房屋、中小水库、尾矿库、中小河流等重点部位和山洪易发区、地质灾害点、城乡易涝区等重点区域巡查防守，及时转移危险区群众，做好基层干部和防汛抢险救援人员安全防护，切实保障人民群众生命安全。要树牢底线思维、极限思维，严防前期受旱地区旱涝急转，细化实化防汛抢险各项措施，全力做好防范应对工作。（总台央视记者 唐国荣）

中央气象台：海南岛广东河北等地将有强对流天气

发表于 2023-05-11 更新于 2025-06-10

2023-05-11 22:15      来源：中央气象台

一、广西海南岛等地出现强对流天气

5月11日8时至14时，广西南部、海南岛东南部等地的部分地区出现短时强降水天气，小时雨量20-60毫米，局地达60毫米以上；另外，内蒙古中部等地的部分地区出现雷暴大风。

二、海南岛广东河北等地将有强对流天气

预计5月11日20时至12日20时，云南中南部、广西东南部沿海、广东西南部、海南岛等地的部分地区将有短时强降水天气，小时雨量30-50毫米，局地可达60毫米以上；另外，内蒙古中部、吉林西部、山西东北部、河北中北部、云南南部等地的局地将有雷暴大风或冰雹天气。

预计5月12日20时至13日20时，辽宁、河北东北部、山东东部、四川南部、云南、广东西南部等地的部分地区将有强对流天气。

预计5月13日20时至14日20时，云南西南部、广东南部沿海、海南岛西北部等地的部分地区将有强对流天气。

强对流天气：目前是指伴随雷暴现象的对流性大风（≥17.2m/s）、冰雹、短时强降水

三、灾害防御建议

1.注意防范雷电可能造成的人员伤亡、设备损失，关注其对航空运输的影响；

2.注意防范短时强降水可能引发山洪、泥石流和城市内涝等次生灾害；

3.注意防范雷暴大风可能造成棚舍和广告牌等搭建物的倒塌以及对高空作业、水域作业和航行的影响；

4.注意防范冰雹可能对农作物和农业设施造成的影响。
